First create a non-battle map!
  • This is a map generator made for RPG games
  • You can generate a new map by pressing Generate map
  • Drag and drop the items on the left side to one of the squares of the map
  • You can click on any square on the map and with the arrow keys you can change the highlighted area
  • If you highlighted a square with an icon on it, you can change the position of the icon with the arrow keys
  • It's possible to open a new battle map with the Generate battle map button and you can get back to the main map with the x button
